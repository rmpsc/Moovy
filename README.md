 ## Getting Started
- Run `flutter pub get`
- Retrieve an API key at [The Movie Database](https://www.themoviedb.org/)
- Locate `lib/services/movie_service.dart` and enter your API key in `final apiKey = 'YOUR API KEY HERE';`  
- You're all set!
